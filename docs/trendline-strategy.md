# 趋势线策略（Trendline Trading）

## 1. 什么是趋势线策略

趋势线策略是一种基于技术分析的交易方法，通过连接价格的关键极值点画出趋势线，
在价格突破趋势线时产生买卖信号。

- **上涨趋势线（支撑线）** — 连接一系列递增的局部低点，形成价格的支撑。价格在趋势线上方运行说明上升趋势完好。
- **下跌趋势线（压力线）** — 连接一系列递减的局部高点，形成价格的压力。价格在趋势线下方运行说明下降趋势延续。

策略的核心思路是**右侧交易**：等待价格确认突破趋势线后再操作，而不是预判方向。

> 趋势线策略的精髓在于**顺势而为**：当下跌趋势被打破时入场，当上涨趋势被打破时离场，用纪律化的规则捕捉趋势转换的机会。

## 2. 关键参数

- **Zigzag 阈值（%）** — 用于识别局部极值的最小波动幅度。阈值越大，过滤掉的噪声越多，检测到的极值点越少。
- **回看窗口（天）** — 每次拟合趋势线时使用的历史数据天数。窗口越长，趋势线越稳定但反应越慢。
- **每笔交易金额** — 每次触发信号时买入的金额，决定了单笔仓位大小。
- **最大持仓笔数** — 同时持有的最大买入笔数，控制总仓位上限，防止过度集中。

## 3. 核心逻辑

策略采用滚动窗口机制，每天重新拟合趋势线并检测突破：

1. 从第 `lookback_days` 天起，每天截取过去 N 天的数据
2. 使用 Zigzag 算法识别局部极值点，拟合上涨/下跌趋势线
3. 将趋势线外推到当日，计算趋势线在当日对应的理论值
4. 对比当日净值与趋势线值，判断突破信号：
   - **突破下跌趋势线** → 下跌趋势可能结束，产生**买入信号**
   - **跌破上涨趋势线** → 上涨趋势可能结束，产生**卖出信号**（清仓）
5. 同一天同方向只交易一次

> 每天重新拟合趋势线意味着趋势线会随着新数据动态调整，比固定趋势线更能适应市场变化。

## 4. 适用场景

- **趋势转换行情** — 市场从下跌转为上涨（或反之）时，趋势线突破信号最有价值。
- **波动较大的品种** — 如行业主题基金、中小盘指数基金，趋势线容易被有效突破。
- **中长期持有** — 回看窗口通常 40~120 天，信号频率低于网格策略，适合中长线操作。

## 5. 注意事项

- **假突破是最大风险** — 价格可能短暂突破趋势线后又迅速回落，导致买在高点或卖在低点。建议配合一定的突破幅度确认（margin_pct）。
- **震荡市频繁交易** — 在无明显趋势的横盘震荡中，趋势线频繁被触碰和突破，会产生过多的无效交易和手续费损耗。
- **窗口长度影响灵敏度** — 窗口太短容易产生假信号，窗口太长可能错过趋势拐点。需要根据品种特性调整。
- **全仓卖出的风险** — 跌破上涨趋势线时策略会清仓，如果是假突破可能导致踏空后续涨幅。

## 6. 参数设置建议

| 风格 | Zigzag 阈值 | 回看窗口 | 每笔金额 | 最大持仓 | 说明 |
|------|-------------|----------|----------|----------|------|
| 保守 | 5.0% | 90天 | 1000元 | 3笔 | 信号少但准确，适合稳健型 |
| 平衡 | 3.0% | 60天 | 1000元 | 5笔 | 兼顾灵敏度与稳定性 |
| 激进 | 1.5% | 30天 | 2000元 | 8笔 | 信号多但假突破风险高，适合活跃品种 |

> 建议先用平衡参数回测，再根据具体基金的波动特性微调。Zigzag 阈值和回看窗口的组合对结果影响最大，可以优先调整这两个参数。
