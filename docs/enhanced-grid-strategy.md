# 增强型网格策略（Enhanced Grid Trading）

## 1. 什么是增强型网格策略

增强型网格策略在经典网格策略的基础上，引入了三个智能优化模块：

- **均线过滤** — 用 N 日均线判断大趋势，只在"顺势"方向交易（低于均线才买、高于均线才卖）
- **ATR 动态间距** — 根据市场波动率（ATR）自动调整网格间距，高波动时拉宽、低波动时收窄
- **递增仓位** — 越低的网格买入金额越大，加速摊薄持仓成本

> 核心思路：在保留网格策略"收割波动"本质的基础上，**让网格更聪明地适应市场环境**，减少逆势操作和无效交易。

## 2. 关键参数

### 基础网格参数（同经典网格）

- **网格上界 / 下界** — 策略运行的价格区间
- **网格数量** — 区间内划分格数（5~20）
- **每格金额** — 基础买入/卖出金额
- **等差 vs 等比** — 网格间距模式
- **底仓** — 首日一次性买入并永久持有的金额

### 增强参数

- **均线过滤周期** — N 日均线的天数（如 20 日）。设为 0 则关闭均线过滤。
  - 净值 < 均线 → 允许买入（市场偏弱，适合抄底）
  - 净值 > 均线 → 允许卖出（市场偏强，适合兑现）
- **ATR 周期** — 计算平均真实波幅的天数（如 14 日）
- **动态网格开关** — 开启后根据当前 ATR 与历史基准 ATR 的比值动态调整网格间距
- **仓位模式** — 固定（每格相同金额）或递增（越低的格子买入越多）
- **递增系数** — 递增模式下每低一格金额乘以此系数（如 1.5 表示每格多买 50%）

## 3. 运作原理

以均线过滤 + 递增仓位为例，假设 20 日均线当前为 1.05，净值区间 0.80~1.20，8 格：

1. 净值 1.10 → 高于均线 1.05 → 净值上穿网格线时**允许卖出**，下穿网格线时**不买入**
2. 净值跌到 0.98 → 低于均线 1.05 → 下穿网格线时**允许买入**，上穿时**不卖出**
3. 净值继续跌到 0.90，触发第 3 格 → 递增仓位：基础 1000 × 1.5² = 2250 元，比固定模式多买 125%
4. 净值反弹回 0.98，重新高于均线 → 上穿网格线时卖出获利

**ATR 动态间距**的效果：
- 市场平静（ATR 偏低）→ 网格收窄 → 更频繁触发，捕捉小波动
- 市场剧烈（ATR 偏高）→ 网格拉宽 → 减少触发频率，避免在大波动中过早耗尽资金

> 三个增强模块可以自由组合：只开均线、只开 ATR、只开递增、或全部开启。建议逐步开启，观察回测效果后再叠加。

## 4. 适用场景

- **宽幅震荡市** — 价格在较大区间内波动，经典网格可能因间距固定而表现不佳，增强型可以自适应
- **趋势不明确的市场** — 均线过滤可以减少"在下跌趋势中追涨"或"在上涨趋势中抄底"的逆势操作
- **波动率变化大的品种** — 如创业板、行业主题基金，ATR 动态间距可以更好地匹配市场节奏

## 5. 注意事项

- **均线过滤可能错过机会** — 如果市场快速反转，均线滞后可能导致错过买入窗口。均线周期越长越稳健但也越滞后。
- **ATR 动态间距需要足够数据** — 前 ATR 周期内的数据用于计算基准，这段时间内策略可能不够稳定。
- **递增仓位加大资金压力** — 越低的格子金额越大，如果市场持续下跌，资金消耗会比固定模式更快。需要确保有足够的备用资金。
- **参数组合多，回测很重要** — 三个增强模块叠加后参数空间变大，建议通过回测找到适合具体品种的参数组合。

## 6. 与经典网格的对比

| 对比维度 | 经典网格 | 增强型网格 |
|--------|--------|---------|
| 买入时机 | 价格下穿网格线即买 | 需同时满足均线条件 |
| 网格间距 | 固定不变 | 可随波动率动态调整 |
| 每格金额 | 固定 | 可递增（越跌买越多） |
| 参数复杂度 | 低（4~5 个） | 中等（8~10 个） |
| 适应性 | 适合窄幅震荡 | 更适合宽幅/变化的市场 |

## 7. 参数设置建议

| 风格 | 均线周期 | 动态网格 | 仓位模式 | 递增系数 | 说明 |
|------|----------|----------|----------|----------|------|
| 保守 | 60日 | 关闭 | 固定 | — | 长周期均线过滤，减少交易次数 |
| 平衡 | 20日 | 开启 | 递增 | 1.5 | 均线 + ATR + 适度递增 |
| 激进 | 10日 | 开启 | 递增 | 2.0 | 短均线 + 强递增，资金需求大 |

> 增强型网格策略的核心理念是**让机械的网格交易获得一定的"市场感知力"**：通过均线判断方向、通过 ATR 感知波动、通过递增仓位优化成本，在保持纪律性的同时提升策略的适应能力。
