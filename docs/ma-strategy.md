# 均线策略（Moving Average）

## 1. 什么是均线

均线（MA, Moving Average）是将一段时间内的收盘价取算术平均后，连成的一条平滑曲线。
它是技术分析中最基础、最常用的指标之一。

计算方式非常简单：

> MA(N) = (第1日收盘价 + 第2日收盘价 + … + 第N日收盘价) ÷ N

常见的均线周期：

- **MA5**（5日均线）— 反映最近一周的平均成本，波动较快，适合短线参考
- **MA15**（15日均线）— 约三周的平均成本，中短期趋势的分界线
- **MA30**（30日均线）— 约一个半月的平均成本，中期趋势的重要参照

## 2. 为什么用均线

基金净值每天波动，短期涨跌充满噪声，直接盯着日净值很容易"追涨杀跌"。
均线的作用在于：

- **平滑噪声** — 把每天的随机波动抹平，呈现出更清晰的趋势方向
- **趋势判断** — 净值在均线之上，说明近期走势偏强；反之偏弱
- **纪律辅助** — 提供客观的"信号"，减少情绪化操作

## 3. 核心信号

### 金叉与死叉

当两条不同周期的均线交叉时，会产生信号：

- **金叉** — 短期均线从下方穿越长期均线，通常被视为看涨信号
- **死叉** — 短期均线从上方穿越长期均线，通常被视为看跌信号

### 价格穿越均线

对于基金定投来说，更实用的信号是净值与均线的关系：

- **净值跌破均线 → 加仓信号**
  当前净值低于均线，说明短期价格低于近期平均成本，相当于"打折买入"
- **净值突破均线 → 减仓信号**
  当前净值高于均线，说明短期已有浮盈，可以考虑适当止盈

> 核心逻辑：**低于平均成本时多买，高于平均成本时少买或止盈**，利用均值回归获取超额收益。

## 4. 在投资帮帮中使用

投资帮帮已内置 MA 策略功能，操作步骤：

1. 登录后进入**我的看板**
2. 添加你关注的基金
3. 点击基金行末尾的 `策略` 按钮
4. 选择均线周期（MA5 / MA15 / MA30）和操作方向（加仓 / 减仓）
5. 保存后，系统会在每个交易日自动计算均线，并在触发信号时提醒你

建议：对于宽基指数基金（如沪深300、中证500），可以从 **MA15 + 跌破加仓** 开始尝试，
周期适中、信号频率合理。

## 5. 注意事项

- **均线具有滞后性** — 均线是对历史数据的平均，无法预测未来。信号出现时，行情可能已经走了一段。
- **震荡市中信号频繁** — 如果基金净值在均线附近反复穿越（"锯齿行情"），会产生大量无效信号。此时应适当放宽判断或暂停策略。
- **不要只看一根均线** — 单一均线容易被"假突破"误导，建议结合多个周期（如 MA5 + MA30）综合判断。
- **均线只是辅助工具** — 任何技术指标都不是万能的。投资决策还需结合基金基本面、市场环境、个人风险承受能力等因素。

> 均线策略的精髓不在于"精准抄底"，而在于**用纪律替代情绪**，在市场低迷时敢于加仓，在市场亢奋时保持冷静。
