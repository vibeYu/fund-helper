# 网格策略（Grid Trading）

## 1. 什么是网格策略

网格策略是一种将价格区间划分为若干等距（或等比）"格子"的量化交易方法。
每当价格下穿一条网格线时买入一份，上穿一条网格线时卖出一份，
从而在震荡行情中反复低买高卖、自动收割波动利润。

可以把它想象成**在价格的阶梯上挂满了渔网**：价格每跌一格就"网"住一份筹码，每涨一格就"抛"出一份筹码。

> 核心思路：**不预测方向，只收割波动**。只要价格在网格范围内来回震荡，策略就能持续获利。

## 2. 关键参数

设计一个网格策略需要确定以下参数：

- **网格上界 / 下界** — 策略运行的价格区间。价格超出区间后策略暂停，避免在极端行情中继续操作。
- **网格数量（格数）** — 区间内划分多少格。格数越多，交易越频繁、单笔利润越小；格数越少则相反。常见设置为 5~20 格。
- **每格金额** — 每次触发网格时买入或卖出的金额（或份额）。决定了单笔交易的仓位大小。
- **等差 vs 等比**
  **等差网格**：每格之间价差相同（如每格间隔 0.05 元），适合价格绝对值较稳定的品种。
  **等比网格**：每格之间涨跌幅相同（如每格间隔 3%），更适合价格波动范围大的品种。

## 3. 运作原理

假设某基金当前净值 1.00 元，设置网格区间 0.80 ~ 1.20，共 8 格（等差，每格 0.05 元）：

1. 净值从 1.00 跌到 0.95 — 下穿第一条网格线，**买入**一份
2. 继续跌到 0.90 — 下穿第二条网格线，**再买入**一份
3. 净值反弹到 0.95 — 上穿网格线，**卖出**之前 0.90 买入的那份（赚 0.05）
4. 继续涨到 1.00 — 上穿网格线，**卖出**之前 0.95 买入的那份（赚 0.05）

每完成一次"买入 → 卖出"循环，就锁定一格的利润。
价格在区间内反复震荡，利润就不断累积。

> 网格策略本质上是**用确定性的小利润换取不确定性的大趋势**。它放弃了"一把赚大钱"的可能性，换来"持续赚小钱"的稳定性。

## 4. 适用场景

- **震荡市（最佳）** — 价格在一个区间内来回波动，网格策略如鱼得水，每次波动都是收益机会。
- **宽基指数基金** — 如沪深300、中证500，长期有底部支撑，不会归零，适合设置网格长期运行。
- **波动率适中的品种** — 波动太小则利润不够覆盖手续费，波动太大则容易突破网格区间。

## 5. 注意事项

- **单边下跌是最大风险** — 价格持续下跌时，网格会不断买入，导致越套越深。必须设置下界止损线，跌破后停止买入。
- **单边上涨会踏空** — 价格持续上涨时，网格会不断卖出，导致过早清仓、错失后续涨幅。可设置上界后保留底仓不卖。
- **手续费不可忽视** — 网格交易频繁，手续费会侵蚀利润。每格利润应显著高于买卖手续费之和（建议至少 3 倍以上）。
- **需要充足的资金** — 网格策略在最坏情况下（价格跌到下界）需要买满所有格子。建议提前计算最大资金占用，避免资金不足导致策略中断。
- **定期检查区间设置** — 市场环境会变化，原来合适的区间可能不再适用。建议每季度或在重大行情变化后重新评估网格参数。

## 6. 与均线策略的对比

- **均线策略**：跟随趋势，适合**趋势行情**，信号较少，单笔盈利空间大
- **网格策略**：收割波动，适合**震荡行情**，交易频繁，单笔盈利小但稳定

两者可以互补：趋势明确时用均线策略顺势操作，方向不明时用网格策略收割震荡。

> 网格策略的精髓在于**承认自己无法预测市场方向**，转而用机械化的纪律从市场的不确定性中提取确定性收益。
